# GAIA — a living planet

A pixel-art, SimEarth-inspired living-planet simulator that runs entirely in the browser: **https://djessemann.github.io/gaia-sim/**

## Editions

- **Pixel** (phone) — the main game: [`/`](https://djessemann.github.io/gaia-sim/)
- **Pixel HD** (iPad landscape): [`/ipad/`](https://djessemann.github.io/gaia-sim/ipad/)
- **Terminal** — the same world drawn entirely in CP437 glyphs and sixteen CGA colours; touch-first, responsive for phone *and* iPad: [`/ascii/`](https://djessemann.github.io/gaia-sim/ascii/)

All editions share the same simulation — terrain, climate, the carbon cycle, evolving life and civilization — with no download and no build step.
